# 本草智瞳小程序简单使用手册

## 一、系统简介

“本草智瞳”是一款面向中药材识别、药材知识查询、智能问答与社区互动的小程序。用户可以通过拍照或上传图片识别药材，也可以检索药材百科资料、查看药材详情、使用智能问答功能获取中药材相关知识，并在社区中发布帖子、评论互动和查看成长任务。

## 二、登录与注册

用户首次进入小程序时，可通过登录/注册页面完成身份认证。
新用户需要填写基本账号信息完成注册，已注册用户可直接登录系统。登录成功后，系统进入首页，用户可以使用药材识别、药材资料查询、智能问答、社区互动和个人中心等功能。

## 三、药材识别功能

用户进入“拍照识药材”模块后，可以选择拍照或从本地相册上传药材图片。图片上传后，系统会调用药材识别服务对图片进行分析，并返回识别结果。

操作流程如下：

1. 进入“拍照识药材”页面；
2. 选择“拍照”或“相册上传”；
3. 上传需要识别的药材图片；
4. 系统自动进行药材识别；
5. 查看识别出的药材名称、识别置信度及相关说明；
6. 可根据需要查看药材详情或保存识别记录。

## 四、药材资料查询功能

用户进入“检索药材百科”模块后，可以通过关键词搜索或分类筛选的方式查找药材资料。系统会展示符合条件的药材列表，用户可点击具体药材查看详细信息。

主要功能包括：

1. 输入药材名称或关键词进行搜索；
2. 按药材分类筛选资料；
3. 查看药材基本信息、功效、禁忌、产地等内容；
4. 查看药材道地地图或相关地理信息；
5. 收藏常用药材资料；
6. 记录或查看药材相关笔记。

## 五、智能问答功能

用户可进入“问答咨询”模块，输入与中药材、药膳、功效、禁忌或使用建议相关的问题。系统会结合上下文信息调用智能问答服务，返回较为完整的回答内容。

操作流程如下：

1. 进入“问答咨询”页面；
2. 在输入框中输入问题；
3. 点击提交问题；
4. 系统生成并展示回答结果；
5. 用户可继续追问、复制回答或分享回答内容。

使用该功能时，建议用户尽量输入清晰、具体的问题，例如“黄芪的主要功效是什么？”“当归和党参可以一起使用吗？”等。

## 六、社区互动功能

用户进入“社区互动”模块后，可以查看社区帖子、发布内容、评论互动，也可以通过签到和完成任务查看成长进度。

主要功能包括：

1. 浏览社区帖子；
2. 发布图文帖子；
3. 对帖子进行点赞、收藏或评论；
4. 查看自己的社区内容；
5. 每日签到；
6. 查看成长等级、任务进度和勋章墙。

发布帖子时，用户可输入文字内容并上传相关图片。系统会对帖子内容进行保存，并在社区页面中展示。

## 七、个人中心功能

用户进入“管理个人信息”模块后，可以查看和修改个人资料，管理收藏内容、浏览记录和观察笔记。

主要功能包括：

1. 查看个人基本信息；
2. 修改头像和昵称；
3. 管理收藏内容；
4. 查看浏览记录；
5. 管理观察笔记；
6. 进行本地与云端数据同步。

## 八、问题反馈功能

如果用户在使用过程中遇到识别错误、资料异常、系统卡顿或其他问题，可以通过反馈入口提交问题说明。系统管理员可在后台查看用户反馈，并对药材数据、图片资源和知识库内容进行维护。

## 九、使用注意事项

1. 上传药材图片时，应尽量保证图片清晰，避免严重模糊、遮挡或光线过暗。
2. 药材识别结果仅供学习和参考，不能直接替代专业医学判断。
3. 智能问答内容仅作为辅助参考，涉及用药安全时应咨询专业人士。
4. 发布社区内容时，应遵守平台规范，不发布无关、虚假或不当信息。
5. 用户应妥善保管个人账号信息，避免账号被他人使用。

## 十、退出系统

用户使用完毕后，可返回个人中心页面进行退出登录操作。退出后，再次进入部分功能时需要重新登录。
